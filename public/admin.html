⚡ API de Integração para N8N / Typebot

Use este endpoint no seu N8N ou sistema externo para criar a instância na EVO, cadastrar no EvoConnect e receber o link de reconexão do cliente automaticamente!

POST https://painel.dmove.com.br/api/v1/auto-create-client
Payload Exemplo (JSON):
{
  "name": "Cliente Restaurante Silva",
  "phone": "5511999998888",
  "instanceName": "silva_whatsapp",
  "createInEVO": true
}
Resposta Automática (JSON):
{
  "ok": true,
  "clientUrl": "https://painel.dmove.com.br/?token=token-silva-xyz"
}
